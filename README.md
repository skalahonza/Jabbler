# Jabbler
Android client for jabber communication protocol.

## Features (planned)
* přihlášení uživatele (+ zapamatování přihlášení)
* připojení do skupiny/místnosti
* šifrování
* ukládání historie lokálně i na server - kdyby umřel mobil
* hovory
* chatování
* smajlíci v chatu
* odhlášení užiatele

## Features (optional)
* videokonference
* posílání souborů
